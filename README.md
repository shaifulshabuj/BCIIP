# Bangladesh Continuous Internet Intelligence Platform (BCIIP)

## Overview
BCIIP is a continuous information collection and intelligence system focused on Bangladesh-related content.

## Setup
1. Install Docker and Docker Compose.
2. Run `docker-compose up --build` to start services.
3. API available at `http://localhost:8000`.
